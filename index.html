About Ghost Chat

Ghost Chat is a completely free, serverless Peer-to-Peer (P2P) messaging application. It allows two users to connect directly via their web browsers using advanced WebRTC technology, ensuring fast, secure, and private communication.

How it Works

  1. One user clicks Create Secure Room to generate a unique 6-character code.
  2. The second user clicks Join Room and enters that exact code.
  3. The browsers connect directly to each other. No messages are ever routed through or stored on a central database.

Frequently Asked Questions

Is my data saved?
No. Because this is a serverless web application, there is no database. Once you close the browser window, the chat history is gone forever.

Do I need an account?
No sign-ups or emails are required. You only provide a local display name that is stored securely on your own device.
